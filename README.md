🚀 Core Features
Real-Time Character Tracking: Dynamic visual feedback for the 200-character limit.

Input Sanitization: Logic-based string manipulation to prevent overflow.

End-to-End Encryption (E2EE) Branding: Visual trust indicators for user privacy.

Responsive Glassmorphism UI: Fully optimized for mobile and high-performance displays (like the Lenovo LOQ).

🛠️ Technical Stack
Frontend: HTML5, CSS3 (Custom Gradients & Backdrop Filters).

Logic: Vanilla JavaScript (ES6+).

Security: Pattern-based input validation.
